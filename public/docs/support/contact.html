In-App Chat ​

Available for Pro and Enterprise customers:

  1. Click chat icon in bottom right
  2. Describe your issue

Emergency Support ​

Enterprise customers only:

Critical system issues:

  • Call emergency hotline (see dashboard)
  • Available 24/7/365
  • Response within 1 hour

Before Contacting Support ​

Help us help you faster:

✅ Check documentation first ✅ Search FAQs for common issues ✅ Try troubleshooting steps ✅ Gather information:

  • Screenshots of error
  • Steps to reproduce
  • Browser and OS version
  • When issue started

What to Include in Support Requests ​

Good Support Request:

Subject: Cannot upload files to report

Issue: When I try to upload a PDF file to a new report,
I get an error message "Upload failed". 

File details:
- Size: 3.2MB
- Type: PDF
- Name: evidence.pdf

Browser: Chrome 120 on Windows 11
Started: Today at 2pm GMT
Already tried: Different browser, smaller file

Screenshot attached.

Poor Support Request:

Subject: Help

It's not working.
