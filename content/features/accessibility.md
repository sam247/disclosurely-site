---

Disclosurely is committed to WCAG 2.1 AA alignment across languages. This page outlines accessibility features, testing coverage, and how to request accommodations.

- Screen-reader friendly UI with semantic HTML and ARIA where needed
- Keyboard navigation support across workflows
- Language tagging via `lang` for localized experiences
- High-contrast UI options and focus states

Need help or have an accessibility request? Contact [support@disclosurely.com](mailto:support@disclosurely.com).
